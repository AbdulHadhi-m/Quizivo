# Quizivo

Quizivo is a modern **MERN stack quiz platform** built for interactive learning, timed challenges, and leaderboard-based competition. Users can register, attempt quizzes, track scores, and compete through real-time rankings.

The platform includes **AI-powered question generation**, enabling dynamic quizzes with category-based customization, difficulty selection, and smarter learning experiences.

Built with **React, Node.js, Express, and MongoDB**, the project focuses on clean UI, secure authentication, timer-based quiz flow, scalable architecture, and intelligent quiz generation.

The **frontend is deployed on Vercel** and the **backend API is deployed on Render**, making the platform fully production-ready with cloud hosting and live accessibility.

## Tech Stack

* React + Vite
* Tailwind CSS
* Redux Toolkit
* Node.js + Express
* MongoDB + Mongoose
* JWT Authentication
* AI-Powered Question Generation
* Vercel (Frontend Deployment)
* Render (Backend Deployment)

# Quizivo

Quizivo is a modern MERN stack quiz platform built for interactive learning, timed challenges, and leaderboard-based competition. Users can register, attempt quizzes, track scores, and compete through real-time rankings.

Built with React, Node.js, Express, and MongoDB, the project focuses on clean UI, secure authentication, timer-based quiz flow, and scalable architecture.

## Tech Stack
- React + Vite
- Tailwind CSS
- Redux Toolkit
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
